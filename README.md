# Project Overview

The Baking Recipes Android app, was made as part of Udacity's Android Developer Nanodegree Program. It allow Udacityâ€™s resident baker-in-chief, Miriam, to share her recipes with the world. The user can select a recipe and see video-guided steps for how to complete it. This app has adaptive UI for phone and tablet devices. Also the user has the ability to add a recipe as a home screen widget.
